# 古典诗词意象→生图提示词实例库

> 积累时间：2026-07-12
> 用途：用户给诗句/意境关键词生图时参考，并持续积累新实例

## 扩写方法

1. 诗句→意象拆解（关键视觉元素）
2. 意象→场景描写（环境、光影、氛围）
3. 场景→风格锚定（画派+技法+色调）
4. 拼装完整提示词

## 实例：碧海青天夜夜心

**原诗**：嫦娥应悔偷灵药，碧海青天夜夜心

**意象拆解**：嫦娥、月宫、碧海、青天、孤独

**完整提示词**：
```
请生成一张16:9比例的中国古典神话工笔画。画面主体：一位中国古代仙女，面容清冷绝美，身穿月白色广袖长裙，衣袂飘飘。她独自站在月宫的玉阶上，仰望碧海青天。环境：月宫清冷，远处是无边的碧海与青天相连，月光如水洒落。光影：冷色调月光从上方倾泻，营造孤寂清冷氛围。整体风格：工笔重彩与水墨写意结合，淡彩设色，大面积留白与雾气交融，古朴诗意，高画质。画面中不要出现任何文字。
```

**生成结果评估**：✅ 意境到位，嫦娥形象清冷孤寂，碧海青天层次分明

## 实例：蒹葭苍苍

**原诗**：蒹葭苍苍，白露为霜，所谓伊人，在水一方

**意象拆解**：芦苇、白露、霜、伊人、水

**完整提示词**：
```
请生成一张16:9比例的中国古典文学插画。画面描绘：深秋清晨，大片枯黄芦苇在晨雾中摇曳，芦苇叶上凝结着白露为霜。一位身着素色衣裙的女子身影隐约出现在水的对岸，若隐若现。环境：宽阔的河面，水雾弥漫，远处天水一色。光影：清晨冷调光线穿透薄雾，营造朦胧迷离氛围。整体风格：水墨写意与现代插画结合，淡彩设色，大面积留白与雾气交融，古朴诗意，高画质，细节极致细腻。画面中不要出现任何文字。
```

**生成结果评估**：✅ 意境朦胧，伊人若隐若现，芦苇与雾气层次丰富
