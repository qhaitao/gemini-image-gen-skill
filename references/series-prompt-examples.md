# 系列图生成实例库

> 积累时间：2026-07-12
> 用途：用户要生成一组主题统一的图时参考

## 分镜设计原则

1. **覆盖叙事弧线**：启程→过程→抵达 / 春→冬 / 起→承→转→合
2. **每张差异在场景和光影**：风格前缀不变，只改画面主体、环境、光影描写
3. **光影变化制造情绪递进**：晨光→正午→夕阳→雪夜

## 统一风格前缀写法

每张提示词都带上相同的风格锚定句（画家+技法+色调+氛围）。

示例："19世纪美国西部风景油画，Albert Bierstadt 的史诗感构图与 Frederic Remington 的人物笔触结合，厚涂技法，色彩浓郁沉稳"
